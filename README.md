# 🌱 Raíces Vivas – Memoria Viva Nicaragua

Aplicación educativa desarrollada en **Kotlin para Android**, cuyo propósito es **preservar, registrar y compartir los saberes populares, culturales y tradiciones de Nicaragua**.  
El proyecto involucra activamente al **sistema educativo, familias y comunidades** como protagonistas, garantizando que el patrimonio cultural e histórico llegue a las nuevas generaciones.

---

## 🎯 Objetivos

- Rescatar y difundir relatos, costumbres y tradiciones de las comunidades nicaragüenses.  
- Fomentar la identidad, soberanía cultural y valores patrios.  
- Crear un espacio colaborativo para estudiantes, docentes, familias y comunidades.  
- Usar la tecnología como herramienta de memoria y educación cultural.  

---

## 🚀 Funcionalidades

1. **Publicación colaborativa de relatos comunitarios**  
   - Formatos: texto, imagen, audio y video.  

2. **Mapa interactivo de memorias**  
   - Relatos geo-localizados por barrio, municipio o comunidad.  

3. **Calendario cultural**  
   - Registro de fiestas patronales, ferias y eventos tradicionales.  

4. **Biblioteca colaborativa**  
   - Saberes populares, recetas, costumbres y artesanías.  

5. **Módulo de retos y juegos didácticos**  
   - Actividades interactivas para aprender sobre identidad y valores patrios.

## 🛠️ Tecnologías Utilizadas

- Lenguaje: **Kotlin**
- IDE: **Android Studio**
- Base de datos: **Firebase / Firestore**
- Mapa interactivo: **Google Maps API**
- UI: **RecyclerView + CardView**
- Almacenamiento multimedia: **Firebase Storage**

## 🌍 Público Objetivo

- **Estudiantes** → conocer y valorar la cultura nacional.
